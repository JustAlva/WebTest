3.表单验证

  • 表单标签 form

    < form action="javascript_data.html" onsubmit="return form_test(this)" target="_blank">
    表单能够包含 input 元素，比如文本字段、复选框、单选框、提交按钮等等。
    表单还可以包含 menus、textarea、fieldset、legend 和 label 元素。

  • form 属性

    • action

      URL,规定当提交表单时向何处发送表单数据。

    • method

      post get,规定用于发送 form-data 的 HTTP 方法。

    • target

      _blank _self _parent _top framename ,规定在何处打开 action URL。

      _blank: 在新窗口中打开。
      _self :默认。在相同的框架中打开。
      _parent :在父框架集中打开。
      _top :在整个窗口中打开。
      framename: 在指定的框架中打开。

三丶DOM

1.什么事DOM

文档对象模型 Document Object Model

2.DOM 节点

HTML 文档中所有内容节点:

  • 整个文档是一个文档节点
  • 每个 HTML 元素是元素节点
  • HTML 元素内的文本是文本节点
  • 每个 HTML 属性是属性节点
  • 注释是注释节点

3.DOM 方法

  • getElementById()

    返回带有指定 ID 的元素。

  • getElementsByTagName()

    返回包含带有指定标签名称的所有元素的节点列表（集合/节点数组）。

  • getElementsByClassName()

    返回包含带有指定类名的所有元素的节点列表。

  • appendChild()

    把新的子节点添加到指定节点。

  • removeChild()

    删除子节点。

  • replaceChild()

    替换子节点。

  • insertBefore()

    在指定的子节点前面插入新的子节点。

  • createAttribute()

    创建属性节点。

  • createElement()

    创建元素节点。

  • createTextNode()

    创建文本节点。

  • getAttribute()

    返回指定的属性值。

  • setAttribute()

    把指定属性设置或修改为指定的值。

4.DOM 属性

  • innerHtml

    获取元素内容

  • nodeName

    节点名称，只读

  • nodeValue

    节点值

  • nodeType

    节点类型，只读

5.DOM 修改

  • 修改 HTML 内容

    document.getElementById("id").innerHTML="new text"

    示例：

  • 改变 HTML 样式

    document.getElementById("id").style.color="color "

    示例：

  • 创建 HTML 元素-appendChild

    //新建子节点
    var para=document.createElement("p");
    var node=document.createTextNode("This is new.");
    para.appendChild(node);
    //给父节点添加新建的子节点
    var parent = document.getElementById("parent_id");
    parent.appendChild(para);

    示例：

  • 创建 HTML 元素-insertBefore

    //新建子节点
    var para=document.createElement("p");
    var node=document.createTextNode("This is new.");
    para.appendChild(node);
    //新建的子节点插入在父节点指定的子节点的前面
    var parent = document.getElementById("parent_id");
    var child = document.getElementById("child_id");
    parent.insertBefore(para,child);

    示例：

  • 删除 HTML 元素

    //将父节点中指定的子节点去除
    var parent=document.getElementById("parent_id");
    var child=document.getElementById("child_id");
    parent.removeChild(child);
    或者
    var child=document.getElementById("child_id");
    child.parentNode.removeChild(child);

    示例：

  • 替换 HTML 元素

    //将父节点中指定的子节点替换
    var para = document.createElement("span");
    var node = document.createTextNode("新替换的内容")
    para.appendChild(node);
    var parent = document.getElementById("parent_id");
    var child = document.getElementById("child_id");
    parent.replaceChild(para,child);

2.window location

  • window location

    用于获取当前页面的地址（url），并把浏览器重定向到新的页面

  • 相关属性

    • location.hostname : web 主机的域名

    • location.pathname : 当前页面的路径和文件名

    • location.port : web 主机端口

    • location.protocol : 当前页面使用的 web 协议

    • location.href : 当前页面 URL

    • location.assign() : 加载新的文档

3.window history

  • window history

    浏览器历史

  • 相关方法

    • history.back() - 与在浏览器点击后退按钮相同

    • history.forward() - 与在浏览器中点击按钮向前相同

4.window navigator

  • window navigator

    访问者浏览器信息

  • 属性

    • appCodeName -- 浏览器的代码名。

    • appName -- 浏览器的名称。

    • appVersion -- 浏览器的平台和版本信息。

    • cookieEnabled -- 浏览器中是否启用 cookie 的布尔值。

    • platform -- 浏览器的操作系统平台。

    • userAgent -- 由客户机发送服务器的 user-agent 头部的值。
