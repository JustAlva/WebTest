  • 动态数据类型

    JavaScript 拥有动态类型。 这意味着相同的变量可用作不同的类型。

    例如:
    var x // x 为 undefined
    var x = 6; // x 为数字
    var x = "Bill"; // x 为字符串

  • 对象

    javascript 中所有事物都是对象

    对象是拥有属性和方法的数据

    创建对象

    定义属性:

    person = new Object();
    person.firstName = "Bill";
    person.lastName = "Gates";

    在构造器中定义方法:

    function person(name,age) {

    this.name = name;

    this.age = age;

    this.changeAge = changeAge;

    function changeAge(age) {

    this.age=age;

    }

    }
    调用:
    person = new person("Bill","50");
    person.changeAge(59);
    document.write(person.name + person.age);

    结果:

    直接定义:

    var customer={fname:"Bill",lname:"Gates",age:56};

    document.write(customer.fname +","+ customer.lname +","+ customer.age);

    结果:

    循环遍历对象属性:

    var text = ''; for (x in customer) { text +=customer[x]+";"; } document.write(text);
