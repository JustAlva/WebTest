4. 样式 style 键值对 以 ; 隔开

  • 字体：font-family:verdana
  • 字体色：color:red
  • 字体大小：font-size:20px
  • 对齐：text-align:center

三丶标签

引用标签

    缩略词 abbr
  • 用法：< abbr title="Hyper Text Markup Language">HTML< /abbr>
  • 样式：HTML

    短引用 q
  • 用法：< q>构建人与自然和谐共存的世界。< /q>
  • 样式：构建人与自然和谐共存的世界。

    长引用 blockquote
  • 用法：< blockquote cite="http://www.baidu.com">这是个长引用< /blockquote>
  • 样式：
    这是个长引用

    定义 dfn
  • 用法：< dfn title="World Health Organization">WHO< /dfn>
  • 样式：WHO

    联系信息 address
  • 用法：
    < address>
    Written by Donald Duck.
    Visit us at:
    Example.com
    Box 564, Disneyland
    USA
    < /address>
  • 样式：
    Written by Donald Duck.
    Visit us at:
    Example.com
    Box 564, Disneyland
    USA

    著作标题 cite
  • 用法：< cite>The Scream< /cite> by Edward Munch. Painted in 1893.
  • 样式：The Scream by Edward Munch. Painted in 1893.

    双向重写 bdo
  • 用法：< bdo dir="rtl">This text will be written from right to left< /bdo>
  • 样式：This text will be written from right to left

四丶HTML样式

1.外部样式表

< head>
    < link rel="stylesheet" type="text/css" href="mystyle.css">
< /head>


2.内部样式表

    < style type="text/css">
        body {background-color: red}
       p {margin-left: 20px}
    < /style>
< /head>
